## О проекте

Данная работа посвящена исследованию инвестиционной привлекательности регионов Российской Федерации с применением методов многомерного статистического анализа — факторного анализа (метод главных компонент) и кластерного анализа (k-means).

В качестве объекта детального анализа выступает **Санкт-Петербург**: исследуется динамика ключевых социально-экономических показателей за период 2000–2023 гг.

## Данные

Файл `data/spb_database.xlsx` содержит несколько листов:

| Лист | Содержимое |
|---|---|
| `спб` | Временной ряд показателей Санкт-Петербурга (2000–2023) |
| `Лист1` | Исходные данные по 36 показателям |
| `Лист3` | Данные после преобразований для стационарности |
| `кластерный 2023` | Данные по 85 регионам РФ за 2023 год (для кластеризации) |
| `Лист2` / `Лист7` | Расшифровка переменных |
| `Лист4` | Результаты тестов на стационарность (ADF) |
| `Лист5` | Корреляционная матрица |
| `Лист6` | Нагрузки главных компонент |
| `Лист8` | Описание показателей и единицы измерения |
| `Лист9` | Динамика инвестиционных показателей |
| `info` | Полные названия переменных |

## Методология

### 1. 📈 Анализ временных рядов
динамика инвестиций
темпы роста
структурный анализ
### 2. 🔍 Факторный анализ (PCA)
снижение размерности
выявление латентных факторов
интерпретация компонент
### 3. 🧩 Кластерный анализ
алгоритм k-means
выбор числа кластеров (WSS, Silhouette, Gap)
сегментация регионов


## 📊 Эконометрическое моделирование (Gretl)
### 🎯 Цель

Определить факторы, влияющие на инвестиции в основной капитал.

### 🧩 Модель

Множественная линейная регрессия:

INVₜ = β₀ + β₁·GRPₜ + β₂·EMPₜ + β₃·INNOVₜ + β₄·RNDₜ + β₅·INCₜ + εₜ

### 🧪 Проверки
ADF — стационарность
Durbin–Watson — автокорреляция
Breusch–Pagan — гетероскедастичность
Jarque–Bera — нормальность
VIF — мультиколлинеарность

### 📈 Результаты

ключевые драйверы: ВРП, инновации, НИОКР
модель статистически значима
высокая объясняющая способность

## Запуск

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/kseniafedotova1-jpg/diploma-spb.git
   ```

2. Открыть проект в RStudio.

3. Установить зависимости:
   ```r
   install.packages(c("readxl", "writexl", "tidyverse", "dplyr", "pastecs",
                      "psych", "corrplot", "factoextra", "cluster", "fpc",
                      "dbscan", "NbClust", "clustertend", "hopkins", "MASS",
                      "sf", "tmap", "tmaptools", "leaflet", "ggplot2",
                      "RColorBrewer", "summarytools"))
   ```

4. Обновить путь к данным в скрипте `R/analysis.R`:
   ```r
   # Заменить строку:
   df <- read_excel("/Users/Ксения/Desktop/диплом/мой диплом/кластерный .xlsx")
   # На:
   df <- read_excel("data/spb_database.xlsx", sheet = "кластерный 2023")
   ```

5. Запустить скрипт `R/analysis.R`.

## Основные результаты

- Выделено **2 кластера** регионов РФ по уровню инвестиционной привлекательности.
- Первые 3 главные компоненты объясняют значительную долю дисперсии исходных данных.
- Регионы-доноры (Москва, ЯНАО, НАО) и города федерального значения образуют отдельные группы, что подтверждает их уникальность.
- Санкт-Петербург демонстрирует устойчивую положительную динамику инвестиционных показателей на горизонте 2000–2023 гг.
- Инновации — главный драйвер инвестиций.
